This project is inspired by Yamaha FGDP layout.

Drum kits on your fingertips.

Basic Key-to-Sound Mappings
  • Space: Kick Drum
  • F, J: Snare
  • G, H: Harder Snare
  • R, U: Closed Hi-hat
  • T, Y: Open Hi-hat
  • I: Ride
  • E: Crash
  • S, D, K, L: Cymbals
  • Z, X: Sticks
  • C, V, B, N, M: Toms

Tip: The key-to-sound map layout is symmetrical. One method of playing is to use both hands so you can play the same sound twice quickly and steadily.
